# Отправка формы контакта — что нужно для работы

**Почему заявки приходят только с вашего компьютера?** На хостинге встроенная `mail()` часто работает только в «своих» условиях, поэтому письма от других пользователей не доходят. **Решение:** отправка через SMTP (логин/пароль почты). В `config.php` должны быть заданы `SMTP_HOST`, `SMTP_USERNAME`, `SMTP_PASSWORD`, и на сервере должна быть папка `vendor/` (см. ниже).

## 1. Установить зависимости (обязательно для SMTP)

В корне проекта выполните:

```bash
composer install
```

Должны появиться папка `vendor/` и файл `vendor/autoload.php`. Без них отправка через SMTP не заработает — заявки от других людей не будут доходить.

## 2. Файлы, без которых форма не заработает

На сервере Timeweb в **корне сайта** должны быть:

| Файл / папка | Назначение |
|--------------|------------|
| `send.php` | Обработчик формы, отправка через SMTP |
| `config.php` | SMTP-настройки (его нет в git — создайте на сервере из `config.example.php`) |
| `vendor/` (вся папка) | PHPMailer и автозагрузка; появляется после `composer install` |
| `vendor/autoload.php` | Подключается в `send.php` |
| `contact.html` | Страница с формой (форма шлёт POST на `/send.php`) |
| `assets/js/script.js` | Отправка формы через fetch и показ тоста |

**Итого минимальный набор:**

- `send.php`
- `config.php` (скопировать с `config.example.php`, подставить реальные данные)
- `vendor/` — вся папка целиком (после `composer install`)
- `contact.html`
- `assets/js/script.js`

Остальные файлы (стили, картинки, локали) нужны для отображения страницы, но не для самой отправки.

## 3. Загрузка на Timeweb

- Папку **vendor/** нужно заливать на сервер (она не в `.gitignore`). Либо выполнить `composer install` на хостинге по SSH, если есть доступ.
- Файл **config.php** в репозитории нет (в `.gitignore`). На сервере создайте его вручную из `config.example.php` и укажите реальные SMTP-данные.

## 4. Проверка

1. Локально: `composer install` → открыть `contact.html`, отправить форму → проверить почту.
2. На Timeweb: убедиться, что на сервере есть `send.php`, `config.php` и папка `vendor/` с `vendor/autoload.php`.
