# Глоссарий — единые русские стандарты терминов

Сайт: SHAR Production. Язык интерфейса: русский (ru). Цель: единообразие, премиальный тон, соответствие требованиям к государственному языку РФ.

---

## 1. production

| Поле | Значение |
|------|----------|
| **Preferred Russian UI term** | производство |
| **Premium alternative** | видеопроизводство (в контексте видео); кинопроизводство (в контексте кино) |
| **Когда допустимо оставить EN** | Только в составе названия бренда: **SHAR Production** (без перевода). В тегах, описаниях должностей и тарифов — только русский вариант. |
| **Brand / proper noun** | Да — только как часть названия компании «SHAR Production». Во всех остальных контекстах (Full Production, AI-Production, «продакшн») — нарицательный термин, переводится. |

**Примеры на сайте:** логотип «SHAR PRODUCTION», тариф «Full Production», тег «AI-Production / Food» → везде, кроме названия студии: «полное производство», «производство с ИИ», «видеопроизводство».
